AUTOMATED WASTE SEGREGATOR

Programming language used: Embedded C

The purpose of this project is the realization of compact, low cost and user friendly segregation system for urban households to streamline the waste management process.

An attempt was made made to develop a system which would help in segregation of waste into dry, wet and metallic waste at household level which will help in the recycling of waste.

Made with love.
